# RimWorld Mods Translations

Сборный репозиторий русских переводов RimWorld-модов.

Каждая папка в корне репозитория является отдельным модом-переводом. Для ручной установки скопируйте нужную папку в `RimWorld/Mods` и включите ее после оригинального мода и Harmony.

## Моды

| Папка | Перевод | Оригинальный мод |
| --- | --- | --- |
| `Empire_Refactored_RU_Full` | Empire Refactored - полный русский перевод | Empire Refactored, Steam ID `3701480464` |
| `Isekai_RPG_Leveling_RU_Full` | ISEKAI RPG LEVELING - Full Russian Translation | ISEKAI RPG LEVELING, Steam ID `3657580708` |
| `Nanotech_Overpower_RU_Full` | [HYE] Nanotech Overpower - полный русский перевод | [HYE] Nanotech Overpower, Steam ID `3533473917` |
| `Plasma_Shield_Implant_RU_Full` | Plasma Shield Implant - полный русский перевод | Plasma Shield Implant, Steam ID `2947388281` |
| `Worksites_Expanded_RU_Full` | Worksites Expanded - полный русский перевод | Worksites Expanded, Steam ID `3687071198` |

Автор переводов: Artas48.
